Networking task_02 Network Devices IP Addressing
 Objective

The purpose of this task is to understand common network devices, IP addressing concepts, and how data travels within a network.

Part A: Network Devices Research
1. Router

Purpose: Connects different networks and directs data packets between them.

How it works: It receives data from one network and forwards it to another using IP addresses.

Real-world usage: Used in homes and offices to connect devices to the internet.

2. Switch

Purpose: Connects multiple devices within the same network.

How it works: It sends data only to the intended device using MAC addresses.

Real-world usage: Used in offices to connect computers, printers, and servers.

3. Hub

Purpose: Connects multiple devices in a network.

How it works: It broadcasts data to all connected devices without checking the destination.

Real-world usage: Previously used in small networks but now mostly replaced by switches.

4. Access Point

Purpose: Provides wireless connectivity to devices.

How it works: It allows Wi-Fi devices to connect to a wired network.

Real-world usage: Used in schools, offices, and public places to provide Wi-Fi.

5. Firewall

Purpose: Protects networks from unauthorized access.

How it works: It monitors incoming and outgoing traffic and blocks suspicious activities.

Real-world usage: Used in homes and organizations for network security.

6. Modem

Purpose: Connects the local network to the Internet Service Provider (ISP).

How it works: It converts digital signals into analog signals and vice versa.

Real-world usage: Installed in homes and offices to provide internet access.
